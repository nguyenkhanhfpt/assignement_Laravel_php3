## Giới thiệu dự án

Project đồ án trường cao đẳng FPT Polytechnic Đà Nẵng, website bán đồ áo quần.

## Công nghệ sử dụng
- Laravel
- Javascript
- Ajax
- SCSS
- Bootstrap

## Yêu cầu
- Composer
- PHP v.7.x
- Mysql v.5.7.x
- Node
- Npm


## Khởi chạy dự án

```
composer install --no-scripts
```

```
cp .env.example .env
```
```
php artisan key:generate
```

```
Cấu hình trong file env
```

```
php artisan migrate
```

```
php artisan db:seed
```

```
php artisan serve
```
